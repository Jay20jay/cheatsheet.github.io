CSS Box Model

CSS Properties Name Description
Width Content Box The content box is an area where content is displayed The content can be changed from inline to block elements.The width and the height of the content can also be altered.
height
padding padding box The padding box sits around the content as white space. A padding of 20px will increase the whitespace around the content.
Margin margin box The margin box is the outermost layer wrapping the the content , padding and border as whitespace. Margin is used to create distance between other sections of a page.
Border border box The border wraps the content and padding. It is usually used to set a frame for the content and padding.
